We The People

Content Supervisor

WE THE PEOPLE is an application that make it easier for citizens to be active in government. The user simply has to input their address and is given access to local officials' information, upcoming bills, and events going on in their area. This full-stack Javascript app uses several APIs, a Bootstrap framework, JQuery, and is deployed using Heroku.
